timezone Cookbook
=================
サーバーのtimezoneの設定を変更します

## Quick start

```
cd /path/to/site-cookbooks
git clone https://kagura.cygames.jp/kimura-ryusuke/chef-timezone.git
mv /path/to/site-cookbooks/chef-timezone /path/to/site-cookbooks/timezone 
```

none

#### packages
